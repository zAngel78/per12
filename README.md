# Sueños de Oro - Sitio Web

Sitio web para la tienda de colchones Sueños de Oro, desarrollado con HTML, CSS y JavaScript.

## Estructura del Proyecto

```
colombo/
├── css/
│   └── styles.css
├── js/
│   └── main.js
├── assets/
│   ├── images/
│   │   ├── logo.png
│   │   └── hero-bg.jpg
│   └── favicon.ico
├── index.html
├── vercel.json
└── README.md
```

## Características

- Diseño responsive
- Navegación intuitiva
- Barra de búsqueda
- Menú de categorías
- Sección Hero con llamada a la acción
- Optimizado para SEO

## Desarrollo Local

1. Clona el repositorio:
   ```bash
   git clone https://github.com/tu-usuario/colombo.git
   ```

2. Abre el archivo `index.html` en tu navegador

3. Para desarrollo, se recomienda usar un servidor local como Live Server de VS Code

## Despliegue

El proyecto está configurado para ser desplegado en Vercel. Para desplegar:

1. Sube el código a GitHub
2. Importa el proyecto en Vercel
3. Vercel detectará automáticamente la configuración y lo desplegará

## Personalización

- Las imágenes se encuentran en la carpeta `assets/images/`
- Los estilos están en `css/styles.css`
- La funcionalidad JavaScript está en `js/main.js`
